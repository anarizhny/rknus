# QA-S2-01 — Тест-план popup (ручной)

## Окружение
- Chrome 120+ (или последняя стабильная версия)
- Расширение загружено в режиме разработчика (chrome://extensions → Developer mode → Load unpacked)

## Чеклист

### Установка
- [ ] Расширение устанавливается без ошибок
- [ ] Иконка появляется в панели расширений
- [ ] Manifest корректен — нет предупреждений на странице chrome://extensions

### Состояние empty-db (первый запуск)
- [ ] При первом запуске popup показывает состояние "empty-db"
- [ ] Отображается прогресс-бар загрузки базы
- [ ] Прогресс-бар анимирован / обновляется
- [ ] Текст указывает на процесс загрузки базы

### Переход к состоянию clean/blocked
- [ ] После завершения загрузки базы popup автоматически переходит к отображению статуса
- [ ] Если текущий домен не заблокирован — показывается состояние clean
- [ ] Если текущий домен заблокирован — показывается состояние blocked

### Заблокированный домен
- [ ] Красный индикатор (иконка или фон)
- [ ] Текст "ЗАБЛОКИРОВАН" (или аналогичный)
- [ ] Отображаются детали: домен, дата блокировки, причина, номер решения
- [ ] Badge на иконке расширения обновляется

### Чистый домен
- [ ] Зелёный индикатор (иконка или фон)
- [ ] Текст "НЕ ЗАБЛОКИРОВАН" (или аналогичный)
- [ ] Детали блокировки не отображаются

### Кнопка "Обновить базу"
- [ ] Кнопка видима и кликабельна
- [ ] При нажатии — появляется спиннер / индикатор загрузки
- [ ] Кнопка блокируется на время загрузки (нет повторного нажатия)
- [ ] После успешного обновления — текст "Обновлено" (с галочкой) на 2-3 секунды
- [ ] После таймаута — кнопка возвращается в исходное состояние
- [ ] При ошибке обновления — отображается сообщение об ошибке

### Dark theme
- [ ] Все текстовые элементы читаемы на тёмном фоне
- [ ] Индикаторы (красный/зелёный) контрастны
- [ ] Кнопки стилизованы корректно
- [ ] Прогресс-бар виден
- [ ] Нет артефактов или "невидимых" элементов

### Специальные страницы
- [ ] chrome:// страницы — badge пустой (без текста)
- [ ] chrome:// страницы — popup показывает нейтральное состояние (не blocked, не clean)
- [ ] about:blank — badge пустой
- [ ] Страница настроек расширения — не крашится

## Критерии прохождения
Все чекбоксы отмечены. Визуальные дефекты задокументированы скриншотами.
